Game Rules

1. Rock paper scissors can be played in 2 modes:
  • Normal mode
  • Ranking mode (by toggling the slide)
2. Ranking mode can only be activated when user hasn't lost any match.3. In ranking mode, the user can compete by accumulating their wins. Once losing a match, the player will be given the option to save their winning record.4. Top 10 players with best winning records are shown in playing ranking.
